# hirobe_et_al_2026
R code and data for statistical analysis for the following paper:

Hirobe, K., Senzaki, M. 

Contents:

1. Hirobeetal2026.R: R script

2. Hirobeetal2026.Rproj: R project

3. Hirobeetal2026.csv: Dataset 1

4. Hirobeetal2026AcousticAttenuation.csv: Dataset 2
